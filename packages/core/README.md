# @aeic-bot2/core
Monorepo core components.

Contains:
 - Database handler
 - Discord bot handler
 - ExoPlatform loader and cache handler
 - Planning IUT Calais loader and cache handler

## Install
```sh
yarn
```

## Build
Files will be builded to `./dist/`
```sh
yarn build
```

See [`Bot and dashboard back-end configuration`](../../README.md#Bot-and-dashboard-back-end).
