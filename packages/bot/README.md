# @aeic-bot2/bot
Main Discord bot. This is a successor to [rigwild/aeic-bot](https://github.com/rigwild/aeic-bot).

## Install
```sh
yarn
```

## Build
Files will be builded to `./dist/`
```sh
yarn build
```

## Build documentation
```sh
yarn doc
```

## Start
```sh
yarn start
```
